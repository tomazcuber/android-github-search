Package br.com.tomazcuber.githubapi.data.model

Class GetUserResponse

  • All Implemented Interfaces:
    @Serializable() 
public final class GetUserResponse

    Data class representing the response from the GitHub API when fetching a user's details.

    This class is marked as serializable to allow kotlinx.serialization to automatically convert JSON data into an instance of this class.
